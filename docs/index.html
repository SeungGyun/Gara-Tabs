Gara — Tab Manager

Tame your tabs. Auto-group by domain, save sessions as profiles, and restore everything in one click.

View on GitHub

Features

💾

Save & Restore

Capture all open tabs and groups as a named profile. Reload them anytime, even after a restart.

📂

Auto-Group

Instantly organize scattered tabs into color-coded groups by domain. No manual sorting needed.

✋

Drag & Drop Editor

Open the full-page editor to rearrange tabs, rename groups, change colors, and fine-tune your workspace.

⏪

Version History

Made a mistake? Roll back to a previous version of any profile with automatic 2-day history.

🚫

Exclude & Filter

Set rules to exclude domains or URL patterns you never want saved.

🌙

Dark Mode

Seamlessly follows your system theme for comfortable browsing day and night.

🌐

8 Languages

English, Korean, Japanese, Chinese, German, French, Spanish, and Portuguese.

📌

Side Panel

Lives in Chrome's built-in side panel — always one click away, never in your way.

🔒

Private & Local

No account required. No data leaves your browser. Just fast, private tab management.

Built With

React 18 TypeScript Vite + CRXJS Zustand Tailwind CSS dnd-kit Chrome Manifest V3
